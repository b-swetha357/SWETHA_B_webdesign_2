# Prototype Responsive Corporate Website  
Web Technology Assignment – Bootstrap 5


## Project Overview

This project is a minimal responsive front-end corporate website prototype developed using Bootstrap 5 framework via CDN. The website represents a Startup / Business theme and is created for demonstration and presentation to a potential client.

The project focuses on layout design, navigation structure, responsiveness, and client-side form validation.


## Theme Selected

Business / Startup Corporate Website


## Web Pages Included

The website consists of four pages:

1. index.html – Home page
2. about.html – About the company
3. services.html – Services offered
4. contact.html – Contact form with validation

All pages are interconnected using a responsive Bootstrap navigation menu.


## Layout Structure Implemented

Each page includes the following components:

- Navigation Menu (Bootstrap Navbar)
- Header Section
- Main Content Section
- Sidebar (on Home page)
- Footer Section
- Integrated Images (3 JPG files)
- Contact Form with Name and Email validation

The layout is built using the Bootstrap Grid System to ensure responsiveness across different screen sizes (desktop, tablet, and mobile).


## Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (CDN)
- JavaScript (for client-side form validation)
- XAMPP (Apache Server for local deployment)


## Features Implemented

- Responsive design using Bootstrap 5
- Mobile-friendly navigation menu
- Structured layout with header, content, sidebar, and footer
- Maximum 3 JPG images integrated
- Contact form with:
  - Required Name field
  - Required Email field (email format validation)
  - Required Message field
- Local deployment using XAMPP


## Deployment Steps (XAMPP)

1. Copy the project folder into:
   C:\xampp\htdocs\
2. Start Apache server from XAMPP Control Panel.
3. Open browser and navigate to:
   http://localhost/folder_name/index.html
4. Verify all pages and form validation are working correctly.


## Outcome

The project successfully demonstrates responsive front-end development using Bootstrap 5, structured layout implementation, navigation linking, image integration, and form validation.

This prototype can be extended into a full-stack corporate website in future development.
